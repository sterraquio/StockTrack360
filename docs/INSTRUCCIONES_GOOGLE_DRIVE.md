## 📹 INSTRUCCIONES PARA CAPTURAR SCREENSHOTS

**Antes de capturar:**

1. Asegúrate que el dev server esté corriendo:
   ```bash
   npm run dev
   ```

2. Abre en navegador: `http://localhost:3000`

3. Prepara datos siguiendo `lista-pantallazos.md`

**Para cada screenshot:**

- Usa `Shift + Win + S` (Windows) o `Cmd + Shift + 5` (Mac)
- Nombra con formato: `01-login.png`, `02-crear-producto.png`, etc.
- Sube a la carpeta `screenshots/` en Google Drive

**Archivos con el siguiente formato:**
- PNG o JPG
- Mínimo 1280x720px
- Mínimo 150 DPI
